# 🧠 Smart Retail Analytics System

A Machine Learning project that combines **sales forecasting** and **profit classification** using the Superstore dataset to support business decision-making.

---

## 📌 Project Overview

This project aims to analyze retail data and build intelligent models that:

- Predict future sales trends 📈
- Classify orders as profitable or non-profitable 💰
- Provide insights to improve business strategies

---

## 🚀 Features

- 📊 Exploratory Data Analysis (EDA)
- 🔥 Correlation Heatmap
- 📉 Discount vs Profit Analysis
- 📆 Monthly Sales Trend Visualization
- 🤖 Sales Forecasting using Random Forest Regressor
- 🎯 Profit Classification using Random Forest Classifier
- 📊 Confusion Matrix Visualization
- ⚠️ Identification of risky (non-profitable) orders

---

## 🧰 Technologies Used

- Python 🐍
- Pandas

---

## 📂 Dataset

- **Superstore Dataset**
- Contains information about:
  - Sales
  - Profit
  - Discount
  - Category
  - Region
  - Order Dates

---

## 📈 Models Used

### 1. Sales Forecasting
- Model: Random Forest Regressor
- Goal: Predict future monthly sales

### 2. Profit Classification
- Model: Random Forest Classifier
- Goal: Classify orders as:
  - Profitable (1)
  - Non-profitable (0)

---

## 📊 Key Insights

- High discounts negatively impact profitability 📉
- Technology category generates the highest sales 💻
- East and West regions are more profitable 🌍
- The model achieved strong performance in both forecasting and classification tasks
